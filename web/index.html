The Promise
Non-negotiable.

The invariants that define what these distributions are and what they never will be.

I

No external API calls. Ever.

Everything runs on your machine. Your data never leaves. Private by construction, not by policy.

II

The AI is invisible.

No mention of AI, LLM, agents, or models in user-facing output. It's just Linux that understands you.

III

Nothing destructive without confirmation.

Destructive and privileged operations require explicit confirmation. No exceptions, no shortcuts.

IV

Every operation is logged.

Full audit trail: timestamp, input, translated command, output, result. Non-negotiable.

V

You never explain your environment.

System context is always injected. If the distribution doesn't know your environment, that's a bug — not a question for you.

Architecture
What makes Radagon
production-ready.
⊕

Deep system context

Every query is prefixed with a live snapshot: kernel version, loaded modules, running services, open ports, disk health, journal tail, SELinux status. You never explain your environment.

✓

Production permission model

Read ops: instant. Write ops: confirm. Destructive or privileged ops: explicit sign-off, always logged. The permission seam is a first-class architectural component, not an afterthought.

◈

Not a chatbot. Trained on Linux.

A model trained from the ground up on the real sources: man pages, RHEL docs, Arch wiki, CVE advisories, kernel mailing lists, and real incident reports. Not a general model dressed up for the occasion.

man pages RHEL docs CVE advisories kernel mailing lists incident reports
≡

Full audit spine

Every operation logged end-to-end: timestamp, natural language input, translated command, stdout/stderr, exit code. Compliance-ready from day one.

◉

SELinux-aware

Rocky Linux 9 runs SELinux enforcing by default. Radagon understands contexts, booleans, and policy it works with SELinux, not around it.

⚡

Performance is a feature

Simple operations feel instant. The system context layer is optimized for p50 latency on common ops. Latency breaks the illusion so we don't let it.
